# Tic_Tac_Toe_Game
Players take turns clicking on the boxes to mark X or O. The game checks for a winner after each move and displays the result. You can reset or start a new game anytime.
This is a basic Tic Tac Toe game built using HTML, CSS, and JavaScript.

Features
  1. Two-player game (X and O)
  2. Turn-based gameplay
  3. Winner detection
  4. Reset and New Game buttons

Technologies Used
  1. HTML
  2. CSS
  3. JavaScript
